# 🧠 MindShield AI
**Predictive Analytics & AI Mental Wellness Platform for Students**


MindShield AI is a comprehensive, AI-driven mental health triage and regulation platform designed specifically for university students. It combines a conversational AI funnel, predictive machine learning, and hardware-accelerated therapeutic tools into a seamless Progressive Web App (PWA).

## ✨ Core Features
* **Clinical Socratic AI:** Utilizes Google Gemini 2.5 Flash for natural, non-interrogative psychological triage, eliminating the need for sterile clinical forms.
* **Predictive Risk Assessment:** Employs a Scikit-Learn Random Forest model on the backend to quantify depression, anxiety, and burnout risk metrics.
* **Therapeutic Engine:** 7 interactive, science-backed emotional regulation games (including EMDR tracking, Box Breathing, and persistent LocalStorage Gratitude Jars).
* **Automated Clinical Reports:** One-click generation of white-label PDF medical reports using dynamic Chart.js Radar logic.
* **PWA & Offline Ready:** Fully installable mobile experience with Service Worker caching.
* **Enterprise Admin Dashboard:** Real-time mock data visualization for university health administrators.

## ⚙️ Tech Stack
* **Frontend:** HTML5, CSS3 (Glassmorphism UI), Vanilla JavaScript, Chart.js, HTML2PDF
* **Backend:** Python, Flask API
* **AI/ML:** Scikit-Learn (Random Forest), Google AI Studio (Gemini 2.5 LLM)
* **Architecture:** Stateless, Zero-Database Retention (Privacy First)
